Documentation: Editing the View Posts Page

This document provides instructions on how to edit the HTML and CSS of the view-posts.html page in your project.

Overview

This project is a marketplace prototype designed to display stock option chains and detailed option contract data, including implied volatility and other metrics. Below is an overview of the technologies and APIs used in this project:

  • Programming Languages: The project is built using HTML, CSS, and JavaScript.
  • Frontend Frameworks: The project uses amCharts for data visualization, specifically for creating gauge charts and other visual elements.
  • APIs: The project integrates with the Finnhub API to fetch stock option chain data and other financial metrics.
  • Dynamic Content: JavaScript is used to dynamically populate the page with data fetched from the API, such as option details, implied volatility, and call/put volumes.
  • Responsive Design: The project is styled to be responsive and user-friendly across different screen sizes.

For more details on specific sections of the project, refer to the table of contents in the sidebar.

HTML Structure

1. Header Section

The header contains the page title and an "Add Post" button:

<header>
    <a href="index.html" class="logo">View Posts</a>
    <button id="addPostButton" class="hidden">Add Post</button>

To Edit:

  • Change the text inside the <a> tag to update the page title.
  • Modify the href attribute to link to a different page.
  • Update the "Add Post" button text by changing the content inside the <button> tag.

2. Main Section

The main section contains:

  • A filter form for filtering posts.
  • A container for displaying posts.
  • A "Load More" button.

Chart Styling

  • Chart Dimensions: The chart dimensions can be controlled using the `#chartdiv` CSS selector.
  • Color Bands: The color bands for the gauge chart are defined in the `bandsData` array in the JavaScript code.
  • Font Size: The font size for the central label can be adjusted in the `am5.Label` configuration.
  • Animation: The chart animations can be controlled using the `chart.appear()` method.

For more advanced customization, refer to the amCharts documentation.

For gauge on option-details.html

The implied volatility gauge chart (located on the options-details.html page) is created using amCharts. Below are the key styling options you can customize:

  • Clock Hand Size: The size of the clock hand can be adjusted using the following properties in the `am5radar.ClockHand` configuration: 
    
var clockHand = am5radar.ClockHand.new(root, {
    pinRadius: am5.percent(15), // Size of the pin at the center
    radius: am5.percent(80),    // Length of the clock hand
    bottomWidth: 20,            // Width of the clock hand's base
});
  • Number Size: The size of the number displayed on the clock hand can be adjusted using the `fontSize` property in the `am5.Label` configuration: 
    
var label = chart.radarContainer.children.push(
    am5.Label.new(root, {
        fill: am5.color(0xffffff),
        centerX: am5.percent(50),
        textAlign: "center",
        centerY: am5.percent(50),
        fontSize: "4em", // Adjust the size of the number here
    })
);
  • Color Bands: The color bands for the gauge chart are defined in the `bandsData` array in the JavaScript code. Each band has a title, color, and range: 
    
var bandsData = [
    { title: "Low", color: "#54b947", lowScore: 0, highScore: 33 },
    { title: "Medium", color: "#fdae19", lowScore: 33, highScore: 66 },
    { title: "High", color: "#ee1f25", lowScore: 66, highScore: 100 },
];
  • Animation: The chart animations can be controlled using the `chart.appear()` method: 
    
chart.appear(1000, 100);

For more advanced customization, refer to the amCharts documentation.
